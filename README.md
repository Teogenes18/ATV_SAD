# ATV_SAD — Students Performance in Exams

Atividade prática de Introdução à Ciência de Dados (Sistemas de Apoio à Decisão), usando o
dataset [Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams).

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```

O app abre em `http://localhost:8501`.
